## 구현시어려웠던점
배경색 변경 버튼을 만드는 기능은 이전 수업에서 이미 배웠던 내용이어서 비교적 쉽게 구현할 수 있었습니다. 
하지만 GitHub 사이트로 이동하는 버튼을 구현할 때는 Process 함수를 사용해야 했기 때문에 해당 함수의 역할과 사용 방법에 대해 추가로 공부한 뒤 코드를 작성하였습니다.
마지막으로 마우스 커서를 사진 위에 올렸을 때 테두리가 변경되는 기능을 구현하는 과정에서는 처음에 어떤 이벤트를 사용해야 하는지 찾는 데 어려움이 있었습니다. 
이후 PictureBox 컨트롤의 속성창에서 번개 모양 아이콘을 통해 다양한 이벤트 목록을 확인할 수 있다는 것을 알게 되었고
그 중 MouseEnter와 MouseLeave 이벤트를 활용하여 원하는 기능을 구현할 수 있었습니다.
